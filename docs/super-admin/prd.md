# 租户运营平台 — 产品需求文档

> 版本 1.1 | 2026-04-15

## 1. 产品概述

租户运营平台是 DCF 平台的**运营方控制台**，面向平台管理员和运维人员，提供租户生命周期管理、跨租户运营监控、全局配置管理等能力。

### 1.1 用户角色

| 角色 | 职责 | 权限范围 |
|------|------|---------|
| 平台管理员（platform_admin） | 平台全面管控 | 所有平台功能 |
| 平台运维（platform_ops） | 日常运维监控 | 只读访问 + 基础操作 |

### 1.2 与租户管理后台的关系

| 超级管理后台 | 租户管理后台 |
|------------|------------|
| 开通租户、分配资源与配额 | 管理租户内员工、技能、工具 |
| 跨租户监控、告警 | 本租户日志、审计 |
| 全局 AI 模型 Provider | 租户级 AI 预算、调用 |
| 平台用户管理 | 租户内用户管理 |

## 2. 功能清单

### 2.1 租户管理（核心功能）

**P0 — 必须**

| 功能 | 说明 | 验收标准 |
|------|------|---------|
| 租户列表 | 展示所有租户（名称/slug/套餐/状态/实例数） | 支持按状态、套餐筛选 |
| 创建租户 | Drawer 表单：基本信息/联系方式/套餐与配额/功能开关/模型访问/初始管理员 | slug 唯一性校验；14 字段四维度配额按套餐自动填充；可选创建初始管理员 |
| 编辑租户 | 修改名称/套餐/配额/联系方式/功能/模型访问 | 切换套餐时所有配额重填（需确认） |
| 暂停租户 | 冻结租户所有实例 | 暂停后租户用户无法登录、实例不可操作 |
| 激活租户 | 恢复暂停的租户 | 激活后租户恢复正常 |
| 归档租户 | 软删除租户 | 归档后不可恢复，数据保留 30 天后物理清理 |
| 用量统计 | 展示租户资源使用情况 | 实例数/用户数/存储使用量/API 调用量 |

**P1 — 应该有**

| 功能 | 说明 |
|------|------|
| 配额告警 | 租户资源使用超过 80% 时告警 |
| 批量操作 | 批量暂停/激活多个租户 |
| 导出 | 租户列表导出 CSV |

### 2.2 平台用户管理

| 功能 | 说明 | 验收标准 |
|------|------|---------|
| 用户列表 | 展示平台域所有用户 | 显示角色、状态、最后登录时间 |
| 创建用户 | 平台管理员/运维角色 | 密码强度校验 |
| 编辑用户 | 修改角色/状态/密码 | 不可修改自己的角色 |
| 禁用用户 | 软禁用 | 禁用后无法登录 |

### 2.3 运营监控

| 功能 | 说明 | 验收标准 |
|------|------|---------|
| 运营总览 | 租户总数/活跃租户/总实例数/运行中实例/已分配 CPU 与内存 | 卡片展示 |
| 配额利用率 | 逐租户实例/并发/CPU/内存/存储使用百分比 | 进度条 + 三色（绿/黄/红） |
| 健康指标 | 跨租户实例健康状态 | 每租户健康度百分比 |

### 2.4 全局配置

| 功能 | 说明 | 验收标准 |
|------|------|---------|
| OpenClaw 运行时配置 | 全局 System Prompt、模型选择 | 配置快照 + 恢复 |
| 模型 Provider 管理 | API Key 配置、健康检查 | 密钥脱敏显示 |
| 风控规则 | 全局风控策略 | 规则测试、快照、批量操作 |

### 2.5 跨租户审计

| 功能 | 说明 | 验收标准 |
|------|------|---------|
| 审计日志 | 跨租户审计查询 | 支持按租户/时间/操作类型筛选 |
| 导出 | JSON / NDJSON 格式导出 | 单次导出上限 10000 条 |

## 3. 交互规范

### 3.1 布局

延续现有管理后台布局风格：

- **侧栏导航** — 7 个一级菜单项
- **内容区** — 列表页：`section.card > div.table-wrap > table` + 分页
- **详情** — 右侧抽屉（`drawer + drawer-mask` 模式）
- **表单** — `.drawer-form` 内 label + input/select

### 3.2 侧栏菜单

```
租户运营平台
├─ 租户管理
├─ 平台用户
├─ 全局配置
├─ 运营监控
└─ 审计日志
```

### 3.3 品牌标识

- Logo: DCF 渐变图标
- 标题: "租户运营平台"
- 副标题: "Digital Crew Factory — 平台管理控制台"
- 角色标签: "平台管理员" / "平台运维"

## 4. 非功能需求

| 项 | 要求 |
|----|------|
| 安全 | 平台 JWT 与租户 JWT 互不通用；API 中间件严格校验 scope |
| 性能 | 租户列表、监控数据支持分页 |
| 可用性 | 租户运营平台独立登录，租户管理后台故障不影响平台操作 |
| 文件规范 | 单文件不超过 1000 行（遵循项目工程宪章） |
