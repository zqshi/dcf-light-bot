# 管理后台 (Admin Console) — 产品需求文档 (PRD)

> 版本 1.0 | 2026-04-14 | MVP 阶段

## 1. 产品定位

管理后台是 DCF 平台的**治理中枢**，供平台管理员和租户管理员使用，提供：
- 数字员工（租户实例）全生命周期管理
- 共享资产（技能/工具/知识）上报→审核→发布→绑定全链路
- RBAC 权限体系与细粒度门控
- 全量审计日志与可观测仪表盘
- AI Gateway 模型路由与成本管控

用户在此处**治理和运营**，不在此处与数字员工对话——对话发生在 Matrix 客户端或 OpenClaw Client Suite。

## 2. 用户角色与权限

| 角色 | 典型权限 | 操作范围 |
|------|---------|---------|
| **超级管理员** | `admin.*` | 全局所有操作 |
| **平台运营** | `admin.runtime.*`, `admin.employees.read`, `admin.logs.*` | 运营监控、日志审计 |
| **资产审核员** | `admin.skills.*`, `admin.tools.*`, `admin.oss.*` | 技能/工具审批发布 |
| **租户管理员** | `admin.employees.{read,write}` (限本租户) | 本租户实例管理 |
| **只读审计员** | `admin.logs.page.*.read`, `admin.auth.page.*.read` | 日志查看、权限查看 |

## 3. 功能模块

### 3.1 数据统计（着陆页）

| 功能 | 描述 | 验收标准 |
|------|------|---------|
| Hero 卡片 | 实例/资产/审计/权限四类核心指标 | 数据来自 `/api/admin/overview`，实时刷新 |
| 趋势图 | 关键指标时间序列 | 最近 7 天 / 30 天切换 |
| 快速入口 | 跳转到高频操作页面 | 权限门控，无权限不显示 |

### 3.2 平台运营

| 功能 | 描述 | 验收标准 |
|------|------|---------|
| SLO 仪表盘 | 可用性/延迟/错误率指标 | 红绿灯状态 |
| 成本概览 | Token 消耗、API 调用量、模型成本 | 按时间范围统计 |
| 保留策略 | 审计日志保留期可视化配置 | 设置后自动清理 |
| 运行状态 | 实例健康分布、异常实例告警 | 实时数据 |

### 3.3 员工管理（实例管理）

| 功能 | 描述 | 验收标准 |
|------|------|---------|
| 实例列表 | 按关键词/租户/固定会话/状态/部门/岗位筛选 | 6 维筛选即时生效 |
| 详情抽屉 | 查看模式：KPI 卡片 + 完整配置信息 | 点击列表行打开 |
| 编辑表单 | 基础信息 + Runtime 配置 + 治理边界 + 审批策略 | 保存后列表刷新 |
| 治理边界 | 职责范围/禁止边界/KPI/升级规则/熔断规则 | 边界检查自动校验 |
| 审批策略 | 按风险等级 L1-L4 配置审批人数/角色/互异要求 | 表格式编辑 |
| 大模型优化 | 一键将治理边界 + 审批策略优化为 System Prompt | API 调用，结果回填到 System Prompt 字段 |

### 3.4 技能管理

| 功能 | 描述 | 验收标准 |
|------|------|---------|
| 技能统计 | 总量/已生效/待处理数量 | 头部统计条 |
| 技能列表 | 按名称/状态/来源筛选 | 即时筛选 |
| 详情抽屉 | 技能配置、绑定租户、审计记录 | skill-detail-renderer 渲染 |
| 审批流 | 上报→待审→通过/驳回→发布 | 完整状态流转 |
| 绑定管理 | 将共享技能绑定到目标租户 | 绑定/解绑有审计 |
| 调试开关 | 技能调试模式切换 | 权限门控 |

### 3.5 工具管理

| 功能 | 描述 | 验收标准 |
|------|------|---------|
| 工具列表 | MCP 服务清单，按名称/协议/状态筛选 | 即时筛选 |
| 健康检查 | 工具服务探活（HTTP/gRPC） | 状态指示灯 |
| 审批流 | 工具注册→审批→发布 | 独立审批页 tools-approvals |
| 操作 | 创建/更新/删除/检查健康/审批/驳回/回滚/重新提交 | 8 种操作全覆盖 |

### 3.6 AI Gateway

| Tab | 功能 | 验收标准 |
|-----|------|---------|
| 模型路由 | 模型列表、负载均衡配置、路由规则 | 热更新 |
| 模板管理 | Prompt 模板 CRUD、版本管理 | 版本可回溯 |
| 调用追踪 | 请求/响应/耗时/Token 统计 | 按时间范围检索 |
| 成本核算 | Token 消耗 → 成本估算 | 仪表盘 |

### 3.7 通知中心

| 功能 | 描述 | 验收标准 |
|------|------|---------|
| 推送渠道 | 渠道列表、状态管理 | 启用/停用 |
| 通知模板 | 模板 CRUD | 支持变量替换 |

### 3.8 行为日志

| 子页面 | 说明 | 筛选维度 |
|--------|------|---------|
| Agent 日志 | 数字员工行为记录 | actor/模块/操作/时间 |
| 服务日志 | 后端服务运行日志 | 同上 |
| 后台日志 | 管理员操作审计 | 同上 |

共用 `logs.js` + `logs-stats.js`，通过 HTML data 属性区分日志类型。

### 3.9 账号权限

| 子页面 | 功能 |
|--------|------|
| 成员管理 | 成员列表、邀请、停用 |
| 用户管理 | 用户 CRUD、密码重置 |
| 角色管理 | 角色定义、权限分配 |

## 4. 交互规范

### 4.1 通用布局

- 左侧固定侧栏（`sidebar`），根据权限动态渲染导航项
- 右侧主内容区（`content`），包含 `page-head`（标题+统计）→ `card`（功能区）
- 右侧抽屉（`employee-drawer`）用于详情/编辑，带蒙层

### 4.2 组件复用约定

| 场景 | HTML 结构 |
|------|----------|
| 列表 | `section.card > div.section-head + div.filter-toolbar + div.table-wrap > table` |
| 抽屉 | `aside.employee-drawer > .employee-drawer-head + .employee-drawer-body` |
| 表单 | `.policy-grid > div > label + input/select/textarea` |
| 统计条 | `.stat-strip > .stat-item > .stat-label + .stat-value` |
| 工具栏 | `.toolbar > .toolbar-group > label + input/select` |
| 分页 | `.table-pager` |

### 4.3 卡片风格

```css
border-radius: 14px;
border: 1px solid #ededf2;
background: #fff;
padding: 20px 24px;
```

## 5. 非功能需求

| 维度 | 要求 |
|------|------|
| 浏览器兼容 | Chrome/Edge/Safari 最近两个主版本 |
| 无缓存 | `Cache-Control: no-store`，确保权限变更即时生效 |
| 安全 | JWT + RBAC + CSP + 白名单文件服务 |
| 性能 | 无构建步骤，页面加载 <1s（本地） |
| 可维护性 | 单文件 <1000 行，IIFE 模式隔离，依赖注入解耦 |
