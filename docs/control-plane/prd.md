# DCF Control Plane — 产品需求文档 (PRD)

> 版本 1.0 | 2026-04-14 | MVP 阶段

## 1. 产品定位

DCF Control Plane 是**多租户数字员工托管平台**的管理层，负责：
- 租户实例的全生命周期编排（创建/启动/停止/重建/删除）
- 跨租户共享资产（技能/工具/知识）的流转治理
- RBAC 权限体系与审计追踪
- Matrix 渠道运营与消息路由

用户通过 **Matrix 客户端**与数字员工交互，OpenClaw 作为运行时引擎对用户不可见。

## 2. 用户角色

| 角色 | 职责 | 典型操作 |
|------|------|---------|
| **平台管理员** | 全局治理 | 实例编排、资产审批、权限配置、审计检索 |
| **租户管理员** | 租户内治理 | 本租户实例管理、资产上报、日志查看 |
| **终端用户** | 日常使用 | 通过 Matrix 与数字员工对话，无需感知后台 |

## 3. 核心能力（MVP）

### 3.1 租户实例管理

| 功能 | 描述 | 验收标准 |
|------|------|---------|
| 实例列表 | 按租户/状态/渠道/部门筛选 | 支持 6 维筛选 + 关键词搜索 |
| 实例详情 | 右侧抽屉展示配置、状态、治理边界 | 点击即开，数据实时联动 |
| 实例编辑 | 修改名称/部门/岗位/风险等级/审批策略 | 保存后列表刷新 |
| 生命周期 | start/stop/rebuild/delete 操作 | 状态 10 秒内刷新 |
| 治理边界 | 职责/禁止/KPI/升级/熔断规则配置 | 边界检查自动校验 |
| 大模型优化 | 一键将治理边界优化为 System Prompt | API 调用 + 结果回填 |

### 3.2 共享资产流转

| 功能 | 描述 | 验收标准 |
|------|------|---------|
| 技能上报 | 租户运行时上报技能到平台 | 上报后进入待审核队列 |
| 审核发布 | 多级审批 → 发布为共享资产 | 审批链路完整，有审计记录 |
| 绑定分发 | 将共享资产绑定到指定租户 | 绑定/解绑操作有审计追踪 |
| 工具管理 | MCP 工具注册、健康检查、审批 | 健康探活 + 状态可视 |
| 知识管理 | 文档 CRUD、分类、版本、权限 | 版本历史可回溯 |

### 3.3 权限与安全

| 功能 | 描述 | 验收标准 |
|------|------|---------|
| 账号管理 | 用户 CRUD、密码重置 | 完整 CRUD |
| 角色权限 | 角色定义 + 权限分配 + 资源粒度 | 权限变更即时生效 |
| 前端门控 | `data-required-permission` 属性 | 无权限元素自动隐藏/禁用 |
| JWT 会话 | 登录签发、过期刷新 | Token 过期后自动跳转登录 |

### 3.4 审计与可观测

| 功能 | 描述 | 验收标准 |
|------|------|---------|
| 行为日志 | Agent/服务/后台三类日志 | 按 actor/模块/操作筛选 |
| 审计追踪 | 全操作审计，支持 context/entity/action/operator 检索 | 可导出 CSV/JSON |
| 保留策略 | 审计日志保留期配置 | 可视化配置 + 自动清理 |
| 平台指标 | 实例状态、SLO、成本概览 | Prometheus + Grafana 仪表盘 |

### 3.5 Matrix 渠道集成

| 功能 | 描述 | 验收标准 |
|------|------|---------|
| 命令路由 | `!create_agent` 等命令 → 实例操作 | 命令执行后 Matrix 回复确认 |
| Bot 管理 | Bot 在线状态、DM 欢迎消息 | Bot 自动加入邀请 |
| 房间绑定 | Matrix 房间与实例关联 | 后台可见绑定关系 |

### 3.6 AI Gateway

| 功能 | 描述 | 验收标准 |
|------|------|---------|
| 模型路由 | 多模型接入、负载均衡 | 路由配置可热更 |
| 模板管理 | Prompt 模板 CRUD | 版本管理 |
| 调用追踪 | 请求/响应/耗时/Token 统计 | 可按时间范围检索 |
| 成本核算 | Token 消耗 → 成本估算 | 仪表盘可视化 |

## 4. API 契约

完整 OpenAPI 规范见 `docs/shared/contracts/openapi/control-plane-v1.yaml`。

核心端点分组：

| 分组 | 前缀 | 说明 |
|------|------|------|
| 健康 | `/health`, `/status`, `/metrics` | 探针与指标 |
| 认证 | `/api/control/auth/*` | 登录、当前用户、ACL |
| 实例 | `/api/control/instances/*` | 实例 CRUD + 生命周期 |
| 资产 | `/api/control/assets/*` | 上报、审核、发布、绑定 |
| 技能 | `/api/control/skills/*` | 技能上报、共享、绑定 |
| 运行时 | `/api/control/runtime/*` | 实例运行时代理 |
| 审计 | `/api/control/audits/*` | 审计检索、导出 |
| Matrix | `/api/integrations/matrix/*` | 命令 webhook |
| 管理后台 | `/api/admin/*` | BFF 聚合接口 |

## 5. 非功能需求

| 维度 | 要求 |
|------|------|
| 租户隔离 | 实例、存储、配置独立；跨租户仅通过共享资产交互 |
| 安全 | RBAC + JWT + 审计留痕 + CSP + Rate Limit |
| 可观测 | Prometheus 指标 + Grafana 仪表盘 + Alert |
| 持久化 | SQLite（开发）/ PostgreSQL（生产）/ FileStore（兼容） |
| 部署 | Docker + K8s + Helm |
| 测试 | 50 test suites / 185 tests 全量通过 |
