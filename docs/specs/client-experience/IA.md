# IA: 客户端信息架构与页面清单

## 1. 用户端 IA（Matrix 客户端）

### 1.1 主导航
- 会话列表
- 数字员工目录
- 我的任务（会话过滤视图）
- 通知中心（失败/审批/告警）

### 1.2 数字员工卡片
- 员工名称
- 实例状态（running/failed/stopped）
- 租户标识
- 技能标签
- 最近活跃时间
- 进入会话按钮

### 1.3 会话窗口增强
- 顶部状态条：执行中/成功/失败
- 任务时间线：关键动作与时间
- 关联实例ID与审计追踪ID
- 失败场景重试提示

## 2. 管理端 IA（当前系统）

### 2.1 总览
- 实例总数/运行中/异常
- 租户覆盖
- 共享资产总量（技能/工具/知识）
- 审计 24h 事件
- 账号与角色概览

### 2.2 员工管理（实例管理）
- 列表字段：实例ID、名称、租户、Matrix房间、部门/岗位、治理边界、能力沉淀、操作
- 操作：查看、编辑、启动、停止、重建、删除

### 2.3 技能管理
- 统计：技能总量、已生效、待处理
- 操作：审核通过、驳回、发布、绑定租户、回滚

### 2.4 工具管理
- 统计：总量、启用、停用、待处理审批
- 列表字段：名称、协议、地址、启用状态、审批状态、更新时间、操作

### 2.5 行为日志
- Agent 行为日志
- 后台操作日志
- 筛选维度：模块、页面、操作

### 2.6 账号权限
- 用户管理
- 角色管理
