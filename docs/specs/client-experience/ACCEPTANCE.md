# 验收标准: 客户端产品形态

## A. 用户侧（Matrix）
- [ ] 用户可在 Matrix 客户端登录并进入会话。
- [ ] 发送 `!create_agent <name>` 后，系统创建实例并返回卡片信息。
- [ ] 用户可继续在房间中与数字员工交互。

## B. 管理侧（Admin）
- [ ] 总览显示实例/资产/审计/权限四类真实指标。
- [ ] 员工页字段为实例口径（实例ID、租户、Matrix房间等）。
- [ ] 技能页显示已生效与待处理统计。
- [ ] 工具页显示审批状态与待处理审批数量。
- [ ] 日志页能按模块/操作筛选并展示行为摘要。

## C. 稳定性与可回归
- [ ] `npm test -- --runInBand` 通过。
- [ ] `npm run e2e:user` 通过。
- [ ] `npm run check:all` 通过（发布前）。

## D. 访问地址
- Matrix Web（Element）：`http://127.0.0.1:8081`
- 管理后台：`http://127.0.0.1:3010/admin/login.html`
